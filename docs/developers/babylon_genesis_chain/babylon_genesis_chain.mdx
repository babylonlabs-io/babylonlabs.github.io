# Babylon Genesis Chain

Babylon Genesis is a Cosmos SDK chain that is backed by Bitcoin staking. 
It is also a chain that facilitate other BSNs to integrate Babylon staking to their protocol. 
Babylon Genesis itself is also a BSN.

| Component | Technology |
|-----------|------------|
| Consensus | CometBFT (a Tendermint fork) |
| Framework | Cosmos SDK |
| Smart Contracts | CosmWasm |
| Relayer | IBC |
| State Machine | Custom modules for BTC staking, BTC Checkpointing, Finality Validation & Incentive Distribution |
