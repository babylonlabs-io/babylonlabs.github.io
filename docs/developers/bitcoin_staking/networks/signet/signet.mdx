# Bitcoin Signet

Bitcoin Signet is Bitcoin's public test network for safely testing
Bitcoin-related functionality without using real BTC. Babylon Genesis Testnet
(`bbn-test-6`) uses Bitcoin Signet for Bitcoin checkpointing and Bitcoin
staking test flows.

## Network Parameters

| Parameter | Value |
| --- | --- |
| Babylon Genesis chain ID | `bbn-test-6` |
| Bitcoin network | `signet` |
| Babylon version for the published snapshot | `v2.3.1` |
| Snapshot height | `9064` |
