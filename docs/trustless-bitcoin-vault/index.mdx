# Trustless Bitcoin Vault

Welcome to the documentation for **Trustless Bitcoin Vault (TBV)** — Babylon's protocol that brings native Bitcoin into Ethereum DeFi without bridges, wrapped tokens, or third-party custodians.

The open Testnet is for the **Aave v4 lending product built on top of TBV**. TBV itself is the underlying, application-agnostic Bitcoin collateral protocol; Aave v4 is the first DeFi application registered with it. On Testnet you experience the lending product end to end — peg-in BTC on signet, borrow stablecoins or WBTC on Sepolia, repay, and redeem.

:::info Testnet status
TBV is currently running on **Bitcoin signet** and **Ethereum testnet**. Funds involved are test-only and have no monetary value. Documentation is being updated progressively as the Testnet evolves.
:::

## Two layers

**TBV protocol and the DeFi applications on top.** A common point of confusion: **TBV** is the underlying Bitcoin collateral protocol; the **Aave v4 lending** integration is the first DeFi application built on top of it. Most user-facing pages in this section describe how things work specifically within the Aave v4 integration. Protocol-layer pages (How it works, Protocol architecture, Safety & trust assumptions, Protocol actors) describe TBV itself and apply across any future application.

If you are here to **try the Testnet**, start with [Quickstart](./use-for-lending/quickstart.mdx).
If you are here to **understand the protocol**, start with [What is TBV?](./start-here/what-is-tbv.mdx).
