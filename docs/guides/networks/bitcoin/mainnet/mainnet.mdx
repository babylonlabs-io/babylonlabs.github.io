# Bitcoin Mainnet

The Bitcoin mainnet is the primary Bitcoin blockchain network where real Bitcoin (BTC) transactions occur with actual economic value.

## Overview

Bitcoin mainnet is the live production network of Bitcoin, where all real Bitcoin transactions take place. This is the actual Bitcoin blockchain where real BTC with real economic value is transacted.

## Bitcoin Network Features

- **Real Bitcoin**: Actual BTC with real-world economic value
- **Production Network**: The live Bitcoin network handling billions of dollars in value
- **Immutable Transactions**: All transactions are permanently recorded on the blockchain
- **Full Scripting Support**: Complete Bitcoin scripting capabilities for advanced contracts
- **Decentralized Security**: Network secured by thousands of nodes worldwide
- **Proof of Work Consensus**: Secured by Bitcoin's proof-of-work consensus mechanism
