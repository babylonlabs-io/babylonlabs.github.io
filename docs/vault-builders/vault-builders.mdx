# Vault Builders

Build applications and integrations on top of Trustless Bitcoin Vaults.

The Vault Builders section will provide comprehensive documentation for developers integrating with the TBV protocol, including:

## Coming Soon

### TypeScript SDK
- **Getting Started** — Installation, configuration, and your first vault interaction
- **Managers API** — High-level APIs for vault creation, collateral management, and borrowing
- **Primitives API** — Low-level transaction building for custom signing workflows (KMS, HSM)
- **Aave v4 Integration Guide** — End-to-end guide for building on the Aave v4 lending integration

### Smart Contract Integration
- **BTCVaultRegistry** — Core protocol contract interfaces for vault lifecycle management
- **AaveAdapter** — Application entry point for Aave v4 vault operations
- **Event Reference** — Key events to monitor for vault state transitions

### API Reference
- **GraphQL Indexer API** — Query vault states, positions, and historical data
- **Vault Provider Proxy API** — Interact with vault provider infrastructure
